How to get Twitter API credentials?

1. Visit Twitter Developer Portal

2. Create a new app or select an existing one

3. Navigate to "Keys and tokens" section

4. Generate and copy all required credentials

5. Paste them in the form above and save

Note: For production deployments, it's recommended to set these as environment variables instead.
